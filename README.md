# Chat-App 💬

A simple **Client–Server Chat Application** developed using **Java**.  
This project demonstrates real-time communication using Java socket programming.

---

## 📌 Project Description
This Chat-App allows multiple clients to connect to a server and exchange messages in real time.  
It is developed as an academic mini-project to understand networking concepts in Java.

---

## 🛠️ Technologies Used
- Java (Core Java)
- Socket Programming
- HTML
- JSP
- JDBC
- MySQL

---

## 📂 Project Structure
- `ChatServer.java` – Server-side logic
- `ChatClient.java` – Client-side logic
- `ChatServerClientHandler.class` – Handles multiple clients
- `DatabaseManager.java` – Database operations
- `Chat.html` – Chat UI
- `ChatHistory.jsp` – Displays stored messages
